52.Animation(1-qism)

CSS Animation-elementni bir "style"dan boshqa "style"ga asta-sekin bilan o`tgazishni ta`minlaydi Animation istalgan CSS xossasini istalgan marta o`zgartirish imkonini beradi

Animation yaratish uchun sihlatiladigan CSS xossalari va qoidalari

@keyframesanimatsiya stillarni qaysi vaqtda qaysi quymatga o`zgarishini belgilb beradi

Animation name-elementmizni ega bo`ladigan animatsiya nomi. @keyframe uchun ko`rsatilgan animatsiya nomi qiymati ko`rinilishida ishlatiladi

Animation durationanimatsiyani qanday davomiylika ega bo`lishini belgilab beradi. Son ko`rinishdagi qiymatga ega bo`lib olchov birliki s(soniya) yoki ms(millisoniya)larda ko`rsatilishi kerak

Animation delay-animatsiya qachon boshlanishini (kechikishini ) belgilab beradi.Son ko`rinishdagi qiymatga ega bo`lib olchov birliki s(soniya) yoki ms(millisoniya)larda ko`rsatilishi kerak

Animation iteration countanimatsiyamiz necha marta qaytarilishini belgilab beradi.Default qiymati 1ga teng
