Trasitions

Trasition-elementimizning xossa qiymatini ko`rsatilgan vaqt davomida ravon op`zgarishini tamillaydi

Trasition delay-effektimiz qachon boshlanishini (kechikishini)belgilab beradi.Son ko`rinishidagi qiymatga ega bo`lib, o`chov birliki s(soniya)yoki ms (millisonoya)larda ko`rsatilishi kerak

Text-shadow

Text-shadow-bu CSS xussiyatli bo`lib, u matnga soya berish uchun ishlatiladi. ushbu xususiyat yordamida matining orqasida turli xil effektlaryaratish mumkun,masalan soya qo`shish, yorqinlik effektlarini berish yoki 3D ko`rinish xosil qilish

2D Transfrom

Transfrom-veb sahifamizdagi elementlarni aylantirish,siljitish,o`lchovini o`zgartirish,va og`dirish uchun ishlatiladigan CSS xossasi.Uning 2 xil ko`rinish mavjud 2D va 3D

Transfrom turlari

  1. rotate()
  2. scale()
  3. skew()
  4. translate()

Rotate-elmentimizni soat strelkasi yoki unga teskari bo`lgan yo`nalishda buradi(aylantiradi)uchun ishlatiladi. Musbat qiymat soat strelkasi, Manfiy esa teskari yo`nalishda buradi

Translate-elmentimizni hozirtgi joyidan siljitish(surish) uchun ishlatiladi. Musbat qiymat ong va pastga qarab, Mnafiy qiymat esa chap va yuqorga qarab siljitish uchun ishlatiladi.

Scale-elmentimizni o`lchovini oshirish yoki kamaytirish uchun ishlatiladi. 0 bilan 1 orasida berilgan qiymat uning o`lachovini kamaytiradi, 1dan katta qiymat esa o`lchovni oshiradi.

Skew-elementimizni ma`lum bir darajaga (deree)og`dirish uchun ishlatiladi. Manfiy qiymat og`dirish yo`nalishini teskari tomonga o`zgartiradi
